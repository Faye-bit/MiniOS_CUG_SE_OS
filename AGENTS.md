# Project Overview

这是一份大学操作系统课程设计作业。你只需在本机上编写代码并上传至GitHub上即可，我会自行在Ubantu虚拟机上测试并把测试结果反馈给你进行修改。

我在Ubantu上的的测试结果我都会放在`./outputs/`目录下。



# Platform Rules

- 编程语言为Rust；
- 本机系统为MacOS，且配置好了homebrew；
- 目标平台为 Ubuntu Linux；
- 优先使用 POSIX API；



# Coding Rules

- 有清晰的代码风格；
- 完备的注释；
- 规范的变量命名。



# Git Workflow

- 代码仓库：`https://github.com/Faye-bit/MiniOS_CUG_SE_OS.git`
- 每完成一个独立功能后提交一次；
- 每次提交只完成一个明确目标；
- commit message 需要明确；
- 禁止一次性大规模重构；



# Other Rules

- 采用自底向上，逐模块逐函数进行单元测试，撰写简单测试用例；
- 始终维护和完善一份设计文档，记录了你的算法逻辑、程序的运行流程等等；
